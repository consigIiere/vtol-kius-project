# Інтегрована КІУС багатороторного БПЛА літакового типу (VTOL)

Курсовий проєкт з розробки комп'ютеризованої інформаційно-управляючої системи для гібридного безпілотного літального апарату. Проєкт орієнтований на виконання завдань автономного моніторингу навколишнього середовища (виявлення термічних аномалій).

##  Структура проєкту

Репозиторій містить цифрову модель та прототип інтерфейсу наземної станції керування (НСК):

* `app.py` — Математична модель та ядро КІУС (Python). Реалізує алгоритми стабілізації, кінцевий автомат (State Machine) режимів польоту та логіку перехідного режиму (Transition Phase).
* `main.js` — Логіка наземної станції керування. Відповідає за асинхронний обмін телеметрією (Downlink/Uplink) та відображення даних на дашборді.
* `index.html` — Структура HMI-інтерфейсу (Human-Machine Interface) тактичної наземної станції, що включає мапу позиціонування та графіки висоти/швидкості.

##  Основний функціонал (згідно з пояснювальною запискою)
1. **Transition Logic:** Автоматизований плавний перехід між багатороторним (VTOL) та крейсерським (Літак) режимами польоту на основі показників швидкості звалювання.
2. **Failsafe & Hazard Avoidance:** Алгоритми термозахисту (екстрений набір висоти при t > 45°C) та подвійний контур енергетичної безпеки (RTH при 10%, Emergency Land при 0%).
3. **Telemetry & Payload:** Обробка даних просторового стану та імітація роботи оптико-електронного цільового навантаження.
