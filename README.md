````markdown
# Шпаргалка: Запуск КІУС (Локально)

## 🟢 Як запустити сайт (Початок роботи)

1. Відкрий термінал у папці проєкту (`KIUS_Dashboard`).
2. Увімкни віртуальне середовище:
   ```bash
   venv\Scripts\activate
````

*(Зліва має з'явитися зелений напис `(venv)`)*
3\. Запусти сервер:

```bash
python app.py
```

4.  Відкрий у браузері посилання: **https://www.google.com/search?q=http://127.0.0.1:5000/**

-----

## 🔴 Як вимкнути сайт (Кінець роботи)

1.  Відкрий термінал, де зараз "біжать" логи сервера.
2.  Натисни **`Ctrl + C`** на клавіатурі (це зупинить сам сайт).
3.  Вийди з віртуального середовища командою:
    ```bash
    deactivate
    ```
4.  Тепер можна закривати термінал і редактор.

<!-- end list -->

